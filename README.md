# Bank Marketing Analytics System

## Overview
The Bank Marketing Analytics System is an interactive Streamlit-based web application developed to analyze and predict customer subscription to bank term deposits using
machine learning.
The application uses the Bank Marketing Dataset from the UCI Machine Learning Repository to perform exploratory data analysis, data visualization, predictive modelling, and
business intelligence reporting.
The project demonstrates the complete machine learning workflow, from data preparation to deployment of trained classification models within an interactive dashboard.

## Objectives
- Analyse customer and campaign data.
- Explore relationships between customer characteristics and subscription behaviour.
- Build and compare multiple machine learning classification models.
- Predict whether a customer is likely to subscribe to a term deposit.
- Generate business recommendations based on analytical findings.

- Logistic Regression

- Random Forest
- Gradient Boosting
- Support Vector Machine
- K-Nearest Neighbours
- Gaussian Naive Bayes

## Dataset
** Dataset Name **
Bank Marketing Dataset
** Source **
UCI Machine Learning Repository
The dataset contains customer demographic information, campaign details and economic indicators collected from direct marketing campaigns conducted by a Portuguese banking
institution.

Target Variable:
- y
   - 1 = Customer subscribed
   - 0 = Customer did not subscribe
